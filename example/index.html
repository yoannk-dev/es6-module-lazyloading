Lazyloading : Un composant vanilla Javascript léger et accessible

Comment l'utiliser ?

Sur l'image à lazyloader

Initialiser un objet js Lazyload avec une classe de son choix (sur l'ensemble du site, on utilisera cette classe pour lazyloader les images de notre choix)

Effet de loading pour les réseaux lents

On peut ajouter un attribut data-loading="true" à l'élément parent. Avec un peu de css (voir exemple dans le projet), on peut afficher un joli loader full css qui fera très bien le job.
